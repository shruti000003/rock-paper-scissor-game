#Rock, paper, scissor game🎮
A simple and fun Rock-Paper-Scissors game built with Python. This project demonstrates basic programming concepts such as conditional statements, loops, and user input handling. It’s a great beginner-friendly project to understand how logic flows in interactive applications.

#Features ✨
Classic gameplay: Choose Rock, Paper, or Scissors against the computer.

Randomized opponent: The computer makes unpredictable moves using random number generation.

Score tracking: Keeps count of wins, losses, and ties.

Replay option: Play multiple rounds without restarting the program.

#How to Play 🕹️
Run the program in your terminal.

Enter your choice: Rock, Paper, or Scissors.

The computer will reveal its choice.

The winner is decided based on the classic rules:

Rock beats Scissors

Scissors beats Paper

Paper beats Rock
